#### 1. CRA

```sh
$ npx create-react-app to-do-ui
```
#### 2. Install NPM:

```sh
$ npm install
```

#### 3. Install Style Components:

```sh
$ npm install --save styled-components
```

#### 4. Start NPM:

```sh
$ npm start
```
##### Open http://localhost: .... to view it in your browser.
